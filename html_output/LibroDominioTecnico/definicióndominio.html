TÉRMINOS

Nombre del término Definición Referencia Normativa
Punto WIFI Un Punto WIFI, en el contexto del Gobierno Autónomo Descentralizado del Distrito Metropolitano de Quito (GAD DMQ), es un punto de acceso inalámbrico gratuito implementado en espacios públicos para brindar conectividad a internet bajo el Servicio Municipal de Internet Público (SMIP)
Fecha de conexión Es la fecha y hora en que el usuario se conecta a los Puntos WIFI
Sesión Es el numero de veces que se conecta un usuario el transcurso del día
Usuarios Es el conteo de usuario único que se conecta en el día a un Punto WIFI
Encuesta WIFI Una encuesta WIFI es el requerimiento bajo demanda, con preguntas y respuestas determinadas por el solicitante en lapso de tiempo
Validación de usuarios La validación de datos de usuario consiste en verificar que los datos proporcionados por el usuario, cumplan con las reglas, formatos y estructuras predefinidas
